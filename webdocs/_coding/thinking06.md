# 성적 말하기
>> 점수에 따라 성적을 알려준다.    
![]({{ site.baseurl }}/assets/images/scratch/ifelse_05.png){: style="height:350px" .img-thumbnail }    


## 1.알고리즘 생각하기
```
1. 점수를 입력한다.
2. 입력받은 점수( 90 / 80/ 70 / 그외 )에 따라 성적을 알려준다.
```

## 3.도전하기
```
1. 변수 리스트를 이용하여 성적표를 만들어 보자.
2. 여러명의 점수목록을 변수 리스트로 만들어 보자
2. 점수에 따라 성적목록을 변수 리스트로 만들어 보자.
```
